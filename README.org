** Prerequisites

*** Software

You will need [[https://github.com/technomancy/leiningen][Leiningen 2.0]] or above installed.


** Running

To start a web server for the application, run:

#+BEGIN_SRC shell
    lein run
#+END_SRC

To do live front-end coding, run:

#+BEGIN_SRC shell
    lein figwheel

#+END_SRC
